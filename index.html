Manis veidotie projekti

SSH Certificate Generator

Veidlapa SSH sertifikātu ģenerēšanai. Sertifikāts tiek izveidots ar iepriekš izveidotiem datiem vai jaunievadītiem datiem. Sertifikāti tiek ģenerēti ar PHP palīdzību un tiek uzreiz parakstīti.
Izmantotie rīki: HTML, JavaScript, CSS, PHP, OpenSSH, Linode.

API aptauja

Īsa aptaujas veidlapa informācijas glabāšanai un cirkulēšanai caur datubāzi. Veidota lai veiktu ieskatu API darbības principā.
Izmantotie rīki: HTML, JavaScript, CSS, NodeJS, MongoDB, Netlify, HerokuApp.

Alkohola Kalkulators

Web kalkulators alkohola satura asinīs izrēķināšanai. Aprēķini tiek veikti pēc datu ievadīšanas. Datu izvēlne ir plaša, lai nodrošinātu pēc iespējas precīzāku aprēķinu. Aprēķini tiek veikti balstoties uz statistiku alkohola iedarbības pētījumos.
Izmantotie rīki: HTML, JavaScript, CSS, Netlify.

Krāsu hronometrs

Hronometrs veidots pēc skolas uzdevuma prasībām. Skaitļi ik sekundi maina krāsu. Ir iespējas pauzēt, piefiksēt laiku un notīrīt pierakstītos rezultātus. Projekts veidots JavaScript zināšanu pilnveidošanai, precīzāk, darbam ar laiku un OOP (objektu orientēto programmēšanu).
Izmantotie rīki: HTML, JavaScript, CSS, Netlify.
